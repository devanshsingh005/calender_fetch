## Overview

This project provides:
- A date input field
- Backend API to return the day and day of the week from a date

## Tech Stack

- Frontend: React.js
- Backend: RESTAPI

## How to Run

### Backend

```bash
cd backend
pip install -r requirements.txt
app.py
```
### Frontend

```bash
cd frontend
npm install
npm start
```
